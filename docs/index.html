Features

OpenResty Manager provides everything you need to manage your OpenResty server with ease.

Reverse Proxy

Easily configure and manage reverse proxy settings for your websites with a user-friendly interface.

Server Status

Monitor your server's performance in real-time with detailed metrics and visualizations.

Free SSL

Automatically manage SSL certificates with Let's Encrypt integration for secure connections.

Access Control

Secure your applications with powerful access control features to protect your resources.

HTTP Flood Protection

Protect your websites from DDoS attacks with built-in HTTP flood protection mechanisms.

Easy to Use

User-friendly interface that doesn't require deep knowledge of OpenResty or Let's Encrypt.

Get Started in Minutes

Simple installation process to get OpenResty Manager up and running.

Either Or Installation

1 Host Version

sudo bash -c "$(curl -fsSL https://om.uusec.com/installer.sh)"

2 Docker Version

curl https://om.uusec.com/docker.tgz -o om.tgz && tar -zxf om.tgz && sudo bash ./om/om.sh
